      Orthogonal

      WIP

      Just like Vectors, a square matrix A∈Rn×n\boldsymbol{A}\isin\mathbb{R}^{n\times n}A∈Rn×n is orthogonal if all of its columns are orthogonal to each other and are normalized.

      As a result,

      ATA=I=AAT \boldsymbol{A}^T\boldsymbol{A} = \boldsymbol{I} = \boldsymbol{A}\boldsymbol{A}^T ATA=I=AAT

      note: The columns of A\boldsymbol{A}A are referred to as being orthonormal (different from orthogonal)

      In other words, the inverse of an orthogonal matrix is its transpose.

      Question: Why A\boldsymbol{A}A must be a square matrix
      Answer:
      Even if the columns of A∈Rn×m,n≠m\boldsymbol{A}\isin\mathbb{R}^{n\times m}, n\neq mA∈Rn×m,n​=m are orthonormal, I=ATA≠AAT \boldsymbol{I} = \boldsymbol{A}^T\boldsymbol{A} \neq \boldsymbol{A}\boldsymbol{A}^T I=ATA​=AAT Hence the term orthogonal is only used when A\boldsymbol{A}A is square.
      • Operating on a vector with an orthogonal matrix will not change its Euclidean norm: ∥Ax∥2=∥x∥2\lVert \boldsymbol{A}\boldsymbol{x}\rVert_2 = \lVert \boldsymbol{x}\rVert_2∥Ax∥2​=∥x∥2​

      Ranks

      WIP

      Column and Row Ranks

      Suppose there is a matrix A∈Rm×n\boldsymbol{A} \isin \mathbb{R}^{m\times n}A∈Rm×n:

      Column rank
      Let CCC be the largest set of linearly independent column vectors of A\boldsymbol{A}A
      column rank=∥C∥\text{column rank} = \lVert C\rVertcolumn rank=∥C∥
      Row rank
      Let RRR be the largest set of linearly independent row vectors of A\boldsymbol{A}A
      row rank=∥R∥\text{row rank} = \lVert R\rVertrow rank=∥R∥

      (Kolter & Do, 2015)

      Matrix Ranks

      For any matrix A∈Rn×m\boldsymbol{A}\isin\mathbb{R}^{n\times m}A∈Rn×m: column rank=row rank \text{column rank} = \text{row rank} column rank=row rank

      Hence, both ranks are collectively referred as the Rank of A\boldsymbol{A}A, denoted as rank(A)rank(\boldsymbol{A})rank(A)

      Properties of A\boldsymbol{A}A’s rank are:

      1. rank(A)≤min⁡(m,n)rank(\boldsymbol{A}) \leq \min{(m,n)}rank(A)≤min(m,n)
      2. Ais full rank  ⟺  rank(A)=min⁡(m,n)\boldsymbol{A} \quad \text{is full rank} \iff rank(\boldsymbol{A}) = \min{(m, n)}Ais full rank⟺rank(A)=min(m,n)
      3. rank(A)=rank(AT)rank(\boldsymbol{A}) = rank(\boldsymbol{A}^T)rank(A)=rank(AT)
      4. rank(AB)≤min⁡(rank(A),rank(B)∀B∈Rm×prank(\boldsymbol{A}\boldsymbol{B})\leq\min{(rank(\boldsymbol{A}), rank(\boldsymbol{B})}\quad\forall \boldsymbol{B}\isin\mathbb{R}^{m\times p}rank(AB)≤min(rank(A),rank(B)∀B∈Rm×p
      5. rank(A+B)≤rank(A+B)∀B∈Rn×mrank(\boldsymbol{A} + \boldsymbol{B})\leq rank(\boldsymbol{A} + \boldsymbol{B})\quad \forall \boldsymbol{B}\isin\mathbb{R}^{n\times m}rank(A+B)≤rank(A+B)∀B∈Rn×m

      Tensors

      Both of Vector and Matrix could be generalized as a Tensor, which is an algebraic object possessing an order or rank (the dimension of the array) (Linear Algebra for Deep Learning, 2021):

      Vectors 1st order Rn,n∈N\mathbb{R}^n, \quad n\isin\mathbb{N}Rn,n∈N Direction and Magnitude
      Matrices 2nd order Rn×m,n,m∈N\mathbb{R}^{n\times m}, \quad n,m\isin\mathbb{N}Rn×m,n,m∈N Linear map

      Matrix-Matrix product

      WIP

      Suppose A∈Rn×m,B∈Rm×p\boldsymbol{A}\isin\mathbb{R}^{n\times m}, \boldsymbol{B}\isin\mathbb{R}^{m\times p}A∈Rn×m,B∈Rm×p:

      C=AB,withci,j=∑k=1mai,kbk,j \boldsymbol{C} = \boldsymbol{A}\boldsymbol{B}, \quad\text{with}\quad c_{i,j} = \sum_{k=1}^{m}{a_{i,k}b_{k,j}} C=AB,withci,j​=k=1∑m​ai,k​bk,j​

      Properties of Matrix-Matrix product:

      • Non-commutative: AB≠BA\boldsymbol{A}\boldsymbol{B} \neq \boldsymbol{B}\boldsymbol{A}AB​=BA
      • Associative: (AB)C=A(BC)(\boldsymbol{A}\boldsymbol{B})\boldsymbol{C} = \boldsymbol{A}(\boldsymbol{B}\boldsymbol{C})(AB)C=A(BC)
      • Distributive: (A+B)C=AB+AC(\boldsymbol{A}+\boldsymbol{B})\boldsymbol{C} = \boldsymbol{A}\boldsymbol{B} + \boldsymbol{A}\boldsymbol{C}(A+B)C=AB+AC
      Question: Why Matrix-Matrix multiplication is not commutative?
      Answer:

      As aforementioned, Matrices represent linear map functions between two vector spaces. The idea is similar to the composition between functions, which is also not commutative:

      f∘g≠g∘ff\circ g \neq g \circ ff∘g​=g∘f (Linear Algebra for Deep Learning, 2021)

      (Computational Linear Algebra for Coders, 2021)

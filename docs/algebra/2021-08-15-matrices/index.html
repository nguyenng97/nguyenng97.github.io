Tensors

Both of Vector and Matrix could be generalized as a Tensor, which is an algebraic object possessing an order or rank, which is the dimension of the array (Linear Algebra for Deep Learning, 2021):

Tensor object Order (Rank) Sets Represent of
Scalars 0th order N,Z,Q,R,etc.\mathbb{N}, \mathbb{Z}, \mathbb{Q},\mathbb{R},\text{etc.}N,Z,Q,R,etc. Magnitude
Vectors 1st order Rn,n∈N\mathbb{R}^n, \quad n\isin\mathbb{N}Rn,n∈N Direction and Magnitude
Matrices 2nd order Rn×m,n,m∈N\mathbb{R}^{n\times m}, \quad n,m\isin\mathbb{N}Rn×m,n,m∈N Linear map
